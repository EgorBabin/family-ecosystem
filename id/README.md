# ID

Единый сервис аутентификации.

- Хост: `id.<domain>`
- Авторизация: `Yandex` и `Telegram`
- После успешного входа сервис выдаёт одноразовый `auth ticket` и перенаправляет пользователя обратно в `gallery`
- Сессия `id` и сессия `gallery` изолированы по разным cookie
- Публикация наружу идёт через `Traefik` и `Let's Encrypt`
